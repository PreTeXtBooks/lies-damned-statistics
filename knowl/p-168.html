Let’s see if there were any outliers in the test score dataset from Example 1.3.1. We found the quartiles and \(IQR\) in Example 1.5.7, so from the \(1.5\,IQR\) Rule, a data value \(x\) will be an outlier if
\begin{equation*} x<Q_1-1.5\,IQR=70-1.5\cdot18=43 \end{equation*}
or if
\begin{equation*} x>Q_3+1.5\,IQR=88+1.5\cdot18=115\ . \end{equation*}
Looking at the stemplot in Example 1.3.1, we conclude that the data values \(25\text{,}\) \(25\text{,}\) and \(40\) are the outliers in this dataset.
