Suppose we look at the “roll two fair dice independently” experiment from Example Example 4.2.6 and Example Example 4.1.19, which was based on the probability model in Example Example 4.1.19 and sample space in Example Example 4.1.4. Let us consider in this situation the random variable \(X\) whose value for some pair of dice rolls is the sum of the two numbers showing on the dice. So, for example, \(X(11)=2\text{,}\) \(X(12)=3\text{,}\) etc.
In fact, let’s make a table of all the values of \(X\text{:}\)
\begin{align*} X(11) &= 2\\ X(21) = X(12) &= 3\\ X(31) = X(22) = X(13) &=4\\ X(41) = X(32) = X(23) = X(14) &= 5\\ X(51) = X(42) = X(33) = X(24) = X(15) &= 6\\ X(61) = X(52) = X(43) = X(34) = X(25) = X(16) &= 7\\ X(62) = X(53) = X(44) = X(35) = X(26) &= 8\\ X(63) = X(54) = X(45) = X(36) &= 9\\ X(64) = X(55) = X(46) &= 10\\ X(65) = X(56) &= 11\\ X(66) &= 12 \end{align*}
