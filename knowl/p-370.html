If we describe an event \(E\) in words as all outcomes satisfies some property \(X\text{,}\) the complementary event, consisting of all the outcomes not in \(E\text{,}\) can be described as all outcomes which don’t satisfy \(X\text{.}\) In other words, we often describe the event \(E^c\) as the event “not \(E\text{.}\)”
